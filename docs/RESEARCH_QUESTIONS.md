# Research Questions

These are the kinds of questions the graph should eventually help answer. They are written to guide what data deserve ingestion and what relationships are worth modeling.

## Core V1 Questions

- Which companies function as the most centralized chokepoints in the chain?
- Which countries are overrepresented in critical upstream stages?
- Which segments appear most concentrated and least redundant?
- Which firms bridge multiple important stages of the chain?
- Where do policy restrictions or sanctions attach to strategically important nodes?

## Disruption Questions

- What happens if Taiwan foundry capacity is materially constrained?
- Which downstream firms appear exposed to a small number of upstream dependencies?
- Which upstream segments would create the widest blast radius if disrupted?
- Which firms or facilities concentrate risk because they sit on many critical paths?

## Evidence And Credibility Questions

- Which relationships are directly supported by primary sources?
- Which relationships are structural or inferred rather than explicitly disclosed?
- Where is the graph strong enough for conclusions, and where is coverage still thin?

## Questions For A Later Product Version

- How have chokepoints changed over time?
- Which new announcements materially change dependency structure?
- Which companies or countries are becoming more or less systemically important?
- What early signals suggest rising policy or operational risk in a segment?

## Implication For The Build

If a dataset does not help answer one or more of these questions, it should have a high bar for inclusion. V1 should favor data that improve insight quality rather than data volume for its own sake.
